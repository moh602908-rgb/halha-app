# دلّني AI

**الإصدار الحالي: v1.0.0**

حلول عملية للحياة اليومية — تطبيق ويب (PWA) يقدّم أدلة خطوة بخطوة لحل أكثر مشاكل الحياة اليومية شيوعًا، بالعربية، ويعمل بالكامل دون إنترنت بعد أول زيارة.

## نطاق هذا الإصدار (Version 1.0)
- **25 مجالًا** تغطي 129 دليلاً كاملاً (بدون أي دليل "قيد الكتابة" — كل الأدلة جاهزة ومكتملة).
- نافذة **"مساعد دلّني"**: وصول سريع لأكثر الأدلة استخدامًا + بحث فوري في كل الأدلة، بدون ذكاء اصطناعي في هذه المرحلة (سيُضاف في المرحلة الثانية عبر خادم خاص، وليس مفتاح API مكشوف في المتصفح).
- يعمل كتطبيق PWA قابل للتثبيت، مع دعم تصفح دون إنترنت بعد أول زيارة.

## الملفات
- `index.html` — الصفحة الرئيسية وهيكل التطبيق
- `style.css` — التصميم والألوان
- `app.js` — بيانات الأدلة (25 مجالًا) ومنطق التنقل ونافذة المساعد
- `manifest.json` + `icon.svg` — إعدادات PWA (قابلية التثبيت على الهاتف)
- `sw.js` — العمل دون إنترنت + التحكم بتحديث النسخة المعروضة للمستخدم

## ⚠️ تنبيه مهم جدًا لأي تحديث مستقبلي
في كل مرة تُحدَّث فيها ملفات المشروع ثم تُرفع من جديد، **يجب تغيير رقم `APP_VERSION` في أعلى ملف `sw.js`** (مثلاً من `v1.0.0` إلى `v1.1.0`).
بدون هذه الخطوة، سيستمر متصفح كل مستخدم بعرض **النسخة القديمة المحفوظة عنده**، حتى لو نجح الرفع على GitHub وVercel بنجاح تام. هذا هو سبب أي مشكلة سابقة من نوع "رفعت التحديث لكن لم يظهر".

## لا يحتاج أي تثبيت
هذا مشروع HTML/CSS/JavaScript خام بدون أي أدوات بناء (لا Node.js ولا npm مطلوبة لتشغيله أو نشره). يكفي رفع الملفات كما هي إلى الاستضافة.

## إضافة دليل أو مجال جديد لاحقًا
أضف عنصرًا جديدًا داخل مصفوفة `guides` الخاصة بالمجال المناسب في `app.js`، بنفس شكل الأمثلة الجاهزة (`title` + `ready: true` + `steps` + `note` اختياري). لإضافة مجال جديد بالكامل، أضف كائنًا جديدًا لمصفوفة `DOMAINS` بنفس البنية (`id` فريد، `name`، `icon`، `guides`).

## المرحلة الثانية (لاحقًا)
دمج مساعد ذكي فعلي عبر خادم وسيط خاص (بدل استدعاء API من المتصفح مباشرة بمفتاح مكشوف كما كان مخططًا سابقًا)، مع إبقاء نافذة الوصول السريع الحالية كخيار يعمل دائمًا دون إنترنت.
