# حلّها

تطبيق ويب (PWA) يقدّم حلولًا عملية خطوة بخطوة لمشاكل الحياة اليومية، بالعربية.

## الملفات
- `index.html` — الصفحة الرئيسية وهيكل التطبيق
- `style.css` — التصميم والألوان
- `app.js` — بيانات الأدلة ومنطق التنقل بين الشاشات
- `manifest.json` + `icon.svg` + `sw.js` — إعدادات PWA (قابلية التثبيت والعمل دون إنترنت جزئيًا)

## لا يحتاج أي تثبيت
هذا مشروع HTML/CSS/JavaScript خام بدون أي أدوات بناء (لا Node.js ولا npm مطلوبة لتشغيله أو نشره). يكفي رفع الملفات كما هي إلى الاستضافة.
