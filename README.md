# دلّني AI

**الإصدار الحالي: v0.5.0**

حلول ذكية للحياة اليومية — تطبيق ويب (PWA) يقدّم حلولًا عملية خطوة بخطوة ومساعدًا ذكيًا لمشاكل الحياة اليومية، بالعربية.

## الملفات
- `index.html` — الصفحة الرئيسية وهيكل التطبيق
- `style.css` — التصميم والألوان
- `app.js` — بيانات الأدلة (20 مجالًا) ومنطق التنقل والمساعد الذكي
- `manifest.json` + `icon.svg` — إعدادات PWA (قابلية التثبيت على الهاتف)
- `sw.js` — العمل دون إنترنت جزئيًا + التحكم بتحديث النسخة المعروضة للمستخدم

## ⚠️ تنبيه مهم جدًا لأي تحديث مستقبلي
في كل مرة تُحدَّث فيها ملفات المشروع ثم تُرفع من جديد، **يجب تغيير رقم `APP_VERSION` في أعلى ملف `sw.js`** (مثلاً من `v0.5.0` إلى `v0.6.0`).
بدون هذه الخطوة، سيستمر متصفح كل مستخدم بعرض **النسخة القديمة المحفوظة عنده**، حتى لو نجح الرفع على GitHub وVercel بنجاح تام. هذا هو سبب أي مشكلة سابقة من نوع "رفعت التحديث لكن لم يظهر".

## لا يحتاج أي تثبيت
هذا مشروع HTML/CSS/JavaScript خام بدون أي أدوات بناء (لا Node.js ولا npm مطلوبة لتشغيله أو نشره). يكفي رفع الملفات كما هي إلى الاستضافة.
